# 1С:Фреш — промо-лендинг (демо)

> Экспериментальный одностраничный лендинг облачного сервиса аренды 1С-программ по подписке.
> Тёмная тема, фирменная красно-жёлтая гамма, плотная компоновка. Чистые **HTML + CSS + JS**, без сборки и зависимостей.

**🔗 Живое демо → https://keyfire.github.io/fresh-landing/**

---

## О проекте

Учебный демо-макет лендинга в духе «облачная 1С без установки»: запусти за пять минут, работай
из браузера откуда угодно. Делается spec-first через скилл **web-design** — сначала [`DESIGN.md`](DESIGN.md),
потом код.

## Стек

- **HTML + CSS + JS** (vanilla) — без фреймворков, сборщиков и npm-зависимостей.
- Шрифты: **Manrope** (кириллица) + **JetBrains Mono** (Google Fonts).
- Интерактив уровня **L2**: reveal по скроллу, sticky-навигация с размытием, магнитный CTA,
  счётчики, бургер-меню (`IntersectionObserver` + `requestAnimationFrame`).
- Доступность: семантика, фокус-состояния, поддержка `prefers-reduced-motion`, адаптив до 375px.

## Запуск локально

```bash
# вариант 1 — простой сервер
python -m http.server 5577
# открыть http://127.0.0.1:5577

# вариант 2 — просто открыть index.html двойным кликом
```

## Дизайн

- Тёмная тема, фирменный акцент `--accent: #E2231A` + тёплый янтарь `--warm: #F59E0B`.
- Все цвета — **только через CSS-переменные** (без хардкода hex в компонентах).
- Сменить акцент = правка `--accent` / `--accent-hover` / `--accent-soft` в [`styles.css`](styles.css)
  (и зеркально в [`DESIGN.md`](DESIGN.md), раздел 2).
- При изменении токена/правила — держать `DESIGN.md` и код в синхроне.
