# fresh-landing — промо-лендинг 1С:Фреш (демо)

Экспериментальный одностраничный лендинг облачного сервиса 1С:Фреш. Делается через скилл
**web-design** (spec-first): сначала `DESIGN.md`, потом код. Статика, без сборки.

## Стек и запуск
- Чистые **HTML + CSS + JS** (без фреймворков, без зависимостей).
- Превью: `.claude/launch.json` → запуск сервера `fresh-landing` (`python -m http.server 5577`) → http://127.0.0.1:5577.
  Или просто открыть `index.html` двойным кликом.

## Файлы
- `DESIGN.md` — **источник правды** по дизайну (9 разделов). Любую правку стиля начинать здесь, потом отражать в коде.
- `index.html` — разметка: nav · hero · доверие(счётчики+лента) · программы(6) · возможности(6) · 3 шага · тарифы(3) · CTA · футер.
- `styles.css` — токены в `:root` + компоненты (все состояния) + адаптив + `prefers-reduced-motion`.
- `app.js` — интерактив L2: reveal по скроллу (прогрессивный, через класс `.js`), sticky-blur навигации, магнитный CTA, счётчики, бургер.

## Текущий стиль (зафиксирован)
- **Тёмная тема**, фирменная **красно-жёлтая** гамма 1С (акцент `--accent: #E2231A` + жёлтый `--warm: #F59E0B`).
- **Плотная** компоновка (контейнер 1120, секции ~80px, gap 16, карточки 22).
- Уровень интерактива **L2**. Шрифт **Manrope** (кириллица) + JetBrains Mono.
- Сменить акцент = правка `--accent` / `--accent-hover` / `--accent-soft` (+ зеркально в DESIGN.md, раздел 2).

## Конвенции
- **Реальный логотип 1С не использовать** (copyright). Сейчас — плейсхолдер-квадрат «1С» (цвет = `--accent`) + дисклеймер по товарным знакам в футере. Названия «1С», «1С:Фреш» — только как текст.
- Цвета **только через CSS-переменные**, без хардкода hex в компонентах.
- Кириллица обязательна — шрифт с поддержкой кириллицы, line-height ≥ 1.6 в тексте.
- Держать `DESIGN.md` и код **в синхроне** (при изменении токена/правила — править оба).
